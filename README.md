# Rust Pulse Pairing Tool

A lightweight desktop tool to pair with Rust servers via Steam and sync to the cloud.

## Download

Download the latest version from the [Releases](https://github.com/TheShadew/Rust-Pulse-Pairing-Tool-v1.0.0/releases) page.

## Features

- Easy Steam pairing - no more copying server IPs and tokens manually
- Cloud sync - your paired servers sync to your Rust Pulse account
- Access your servers from anywhere on the web

## System Requirements

- Windows 10 or later
- ~100 MB disk space
- Internet connection for pairing

## Usage

1. Download and install the app
2. Click "Pair via Steam" and log in
3. Go to a Rust server, find a computer station, and click "Pair with Server"
4. Your server appears in the app - click "Sync to Cloud" to access it on the web
